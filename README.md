## 📖 About

**Retina Safe** is a collection of interactive, browser-based eye relaxation techniques built with pure HTML, CSS, and JavaScript. In an era of extended screen time, this project provides guided exercises to help users reduce eye fatigue, improve focus, and maintain long-term eye health — all without requiring any installations or dependencies.

---

## ✨ Features

### 🕐 20-20-20 Rule Timer
A guided timer based on the ophthalmologist-recommended **20-20-20 rule**: every **20 minutes**, look at something **20 feet** away for **20 seconds**. This module provides:
- Automated interval reminders
- Visual countdown display
- Session tracking

### 🧘 Calming Stillness
An immersive relaxation experience designed to soothe tired eyes through:
- Ambient visual animations
- Breathing-synchronized guidance
- Calming color palettes optimized for eye comfort

### 🎯 Eye Focus Calibration
Interactive exercises to strengthen eye muscles and improve focus flexibility:
- Dynamic focus-shifting targets
- Guided near-far focus drills
- Progress tracking for calibration sessions

## 🚀 Getting Started

### Prerequisites

All you need is a modern web browser — no installations, build tools, or dependencies required.

### Usage

1. **Clone the repository**
   ```bash
   git clone https://github.com/pratikranjan2212/Retina-Safe-Eye-Care.git
   cd Retina-Safe-Eye-Care
   ```

2. **Open any technique**
   
   Navigate to the desired technique folder and open the `index.html` file in your browser:
   ```bash
   # 20-20-20 Rule Timer
   open relaxation_techniques/20-20-20-rule/index.html

   # Calming Stillness
   open relaxation_techniques/calming-stillness/index.html

   # Eye Focus Calibration
   open relaxation_techniques/eye-focus-calibration/index.html
   ```

   > 💡 **Tip:** On Windows, replace `open` with `start`. On Linux, use `xdg-open`.

## 📋 Recommended Eye Care Practices

| Practice | Recommendation |
|----------|---------------|
| **Screen Breaks** | Follow the 20-20-20 rule consistently |
| **Lighting** | Ensure proper ambient lighting while using screens |
| **Distance** | Maintain at least 20 inches between eyes and screen |
| **Blinking** | Blink frequently to keep eyes moisturized |
| **Blue Light** | Use blue-light filtering after sunset |
